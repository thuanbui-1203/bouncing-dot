# Bouncing Dot

A Unity project featuring shader-based animations and visual effects.

## Description

Bouncing Dot is a Unity project that demonstrates shader programming techniques using ShaderLab and HLSL. The project showcases visual effects and animations created primarily through custom shaders.

## Technologies

- **Unity** - Game engine
- **ShaderLab (78.9%)** - Unity's shader language
- **HLSL (16.2%)** - High-Level Shading Language
- **C# (4.9%)** - Scripting

## Getting Started

### Prerequisites

- Unity Editor (recommended version: check `ProjectSettings/ProjectVersion.txt`)
- Basic understanding of Unity and shaders

### Installation

1. Clone the repository:
```bash
git clone https://github.com/thuanbui-1203/bouncing-dot.git
```

2. Open the project in Unity:
   - Launch Unity Hub
   - Click "Add" and navigate to the cloned repository folder
   - Open the project with the compatible Unity version

3. Open the main scene from the `Assets` folder

### Running the Project

1. Open the project in Unity Editor
2. Navigate to the desired scene in the `Assets` folder
3. Press the Play button to run the project
